## 🚀 What I'm Up To

### Scouty Autonomous Robotics Platform 🤖

Building **Scouty**, a perception driven mobile robot focused on navigation, interaction, and real world autonomy.

**Current focus areas:**

- Sensor integration with Raspberry Pi and microcontrollers
- Lidar based environment awareness
- Vision based tracking and object detection
- Voice interaction pipeline
- Motor control and power system architecture
- Modular embedded firmware design
- Distributed robotics software architecture

**Planned expansions:**

- SLAM based navigation
- Gesture recognition
- Edge deployed ML inference
- Custom PCB control stack
